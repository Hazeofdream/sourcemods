 plugin that makes modifying friendly-fire values around grenade launchers better, and also prevents screenshake and FF notifications against teammates

Supports: Toggling of the plugin

( Convar: gl_adjff_enable )

0 to disable, 1 to enable

Supports: Scaling of self-damage via the grenade launcher

( Convar: gl_adjff_self_scale )

1.0 default

0-1 scale supported.
